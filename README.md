# AI 面部美学测评

一个面诊前参考工具静态页面。用户可扫码进入页面，上传自拍后生成面部美学沟通清单，帮助用户在面诊前整理自己的关注点。

## 功能

- `assets/ai-face-qr.png`：AI 测脸入口二维码，扫码进入网页。
- `assets/wechat-qr.png`：微信号信息二维码，正式使用可替换成微信或企业微信导出的二维码。
- 图片只在浏览器本地预览，当前静态页面不会自动上传或保存照片。

## 合规边界

- 不做医疗诊断。
- 不承诺治疗或变美效果。
- 不制造容貌焦虑。
- 不展示夸张术前术后对比。
- 真实项目建议应由专业人员面诊后确认。

## 本地预览

直接打开 `index.html`，或运行：

```bash
npm start
```
