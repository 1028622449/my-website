# 悦颜增长 AI

医美整形行业的 AI 测脸获客助手静态页面。

## 核心钩子

用户扫码进入 AI 测脸页，上传自拍后生成面部美学沟通清单，再引导一键添加美学顾问微信。

第一版不接后台数据库，照片只在浏览器本地预览。用户完成测脸后，页面弹出微信二维码入口，发送暗号“测脸”领取完整报告。

- `assets/ai-face-qr.png`：AI 测脸入口二维码，扫码进入网页。
- `assets/wechat-qr.png`：微信号信息二维码，正式投放可替换成微信/企微导出的加好友二维码。

## 合规边界

- 不做医疗诊断。
- 不承诺治疗或变美效果。
- 不制造容貌焦虑。
- 不展示夸张术前术后对比。
- 真实项目建议应由专业人员面诊后确认。

## 本地预览

直接打开 `index.html`，或运行：

```bash
npm start
```
